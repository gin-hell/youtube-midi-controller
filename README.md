# yyyyytube.com

web.app for midi-controlled live YouTube performance 

<!-- ![](imgs/big-teen.png) -->

yyyyytube.com is a Chrome web.app originally created for [big TEEN](https://docs.google.com/document/d/17wKxBYxss0mqF_Kv-o9eMVcPbl_DLUBRNLB7ADo97rk/edit?usp=sharing) with/for [Andrew Tham](https://www.google.com/search?q=andrew+tham&source=lnms&tbm=isch&sa=X&ved=0ahUKEwibuNHao-jTAhXC3YMKHcFXDtwQ_AUIDCgD&biw=1280&bih=633) that allows performers to control multiple playback parameters of any video uploaded to YouTube(dot)com via MIDI protocol.

yyyyytube.com uses both the [YouTube API](https://developers.google.com/youtube/) and the [Web MIDI API](https://www.w3.org/TR/webmidi/)
