#BIGTEEN (not the porn thing)

  • use the #BIGTEEN search bar to search for a YouTube video
  • the drop-down menu below the video player will display the top 5 results matching yr search terms
  • click on a playback function icon below to assign a MIDI control ~ reassign a MIDI control simply by clicking again
  • functions are, in order of the icons below:
    • go back 1 video in results
    • skip back 1 second
    • pause
    • play
    • skip forward 1 second
    • go forward 1 video in results
    • trigger loop
    • loop size
    • playback rate
    • volume
  • playback rate, loop size, && volume are all control values from 0 - 127
  • set yr loop size before you set yr loop control to avoid the black hole of infinitely tiny youtubes

move a MIDI control to assign it to the function u just clicked on !
